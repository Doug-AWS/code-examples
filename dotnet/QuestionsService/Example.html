Creating a serverless app in C#

This document walks you through the steps to create a serverless app in C#. We will use the CDK to create the resources, including:

  • An AWS Lambda function
  • An Amazon API Gateway API to call the Lambda function
  • An Amazon S3 bucket that contains the Lambda function code
  • An Amazon DynamoDB table to store our question

Let's get started.

  1. Create a new folder for our CDK project. Let's call it QuestionsService, and navigate into it. 
    mkdir QuestionsService
cd QuestionsService
  2. Run the following command to create an empty CDK project. 
    cdk init --language csharp
  3. Open src\QuestionsService.sln in Visual Studio.
  4. Select Tools, NuGet Package Manager, Manage NuGet Packages for Solution in Visual Studio, select Browse and add the following packages.
    • Amazon.CDK.AWS.ApiGateway
    • Amazon.CDK.AWS.DynamoDB
    • Amazon.CDK.AWS.Lambda
    • Amazon.CDK.AWS.S3
    • Amazon.Lambda.Core
